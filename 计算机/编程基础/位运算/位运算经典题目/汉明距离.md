# 题目
LeetCode第461题，汉明距离

两个整数之间的**汉明距离**指的是这两个数字对应二进制位**不同**的位置的数目

给你两个整数`x`和`y`，计算并返回它们之间的汉明距离
# 思路
根据异或运算的规则，“**同为0，异为1**”，我们只需要将`x`和`y`进行异或运算，结果中1的位置便是`x`和`y`二进制中值相反的地方，只需要统计结果中1的个数便可得到汉明距离
# 代码
```csharp
public int HammingDistance(int x, int y) {
    int result = 0;
    int xor = x ^ y; // x异或y,此时还存在的1，代表x和y在该位上不同
        
    //统计xor中的1的数量即可
    while (xor > 0)
    {
        if ((xor & 1) == 1)
            result++;

        xor = xor >> 1;
    }
    return result;
}
```
