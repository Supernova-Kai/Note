# 线性查找

线性查找，也就顺序查找，其原理是从集合的开头向末尾遍历，直到找到目标值或到达集合末尾为止。

# 代码示例

```c#
public bool LinearSearch(int[] array, int target)
{
    for (int i = 0; i < array.Length; i++)
    {
        if (array[i] == target)
        {
            return true; // 找到了
        }
    }
    return false; // 没找到
}
```

# 复杂度分析

### 时间复杂度

- 最好情况：目标值元素正好在第一个，时间复杂度为 $O(1)$。

- 最坏情况：目标值元素正好在最后一个，或不在集合内，需要遍历完整个集合，其时间复杂度为 $O(n)$。

  综上，时间复杂度取最坏情况，为 $O(n)$。

### 空间复杂度

未开辟新空间，为 $O(1)$。
