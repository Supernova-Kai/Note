## 左移
对于数x，对其左移n位，相当于将其与2的n次方相乘，即：

$$
x << n = x * 2^n
$$

注意，该操作可能会溢出
## 右移
对于数x，对其右移n位，相当于将其与2的n次方相除，即：

$$
x >> n = x / 2^n
$$
## 取二进制中的部分位
由于与运算必须满足“两个比特位均是1，结果才是1”，我们可以利用这一规则，来提取二进制中的任意一位

假设我们要取x的最后一位，我们只需要将x与1进行与运算即可，这是因为：

1. 对于两数二进制的最后一位，由于1的最后一位是1，如果x的最后一位是1，与运算的结果是1；如果x的最后一位是0，与运算的结果是0，也就是说最后一位的结果取决于x的最后一位是0还是1
2. 对于非最后一位，由于1除了最后一位是1，前面的所有位均是0，由于0与0、1进行与运算结果都是0，所有不论x前面的值是0还是1，结果均是0，也就是会被丢弃

注意，用于与运算的数的位数一定要匹配
```csharp
Int16 x = 5;
(x & 1) == 1; // 判断最后一位是否是1
(x & 0b0000000000000010) == 0b0000000000000010; // 判断倒数第二位是不是1
(x & 0b0000000000000011) == 0b0000000000000011; // 判断最后两位是不是均是1

// 也可以先右移再取最后一位
((x >> 1) & 1) == 1 // 判断倒数第二位是否是1
```

## 判断奇偶性
对于奇数，其二进制最后一位一定是1；对于偶数，其二进制最后一位一定是0

我们可以取某个数的最后一位，来判断其奇偶性
```csharp
// 是否是偶数
public bool IsEven(int n)
{
    return (n & 1) == 0;
}
```
## 设置二进制中的指定位为0（置零）
对于与运算，有：

1. `0 & x = 0（x为1或者0）`，也就是0可以用于将原值置零
2. `1 & x = x（x为1或者0）`，也就是1不会改变原值
    
我们可以利用这个特性，用于设置二进制中的指定位甚至多位为0
```csharp
Int16 x = 5;
x = (x & 0b1111111111111110); // 置零x的最后一位
x = (x & 0xFFFE); // 二进制数太长，可以用16进制数表示

x = (x & 0b1111111111111100); // 置零x的最后两位
```
## 设置二进制中的指定位为1
对于或运算，有：

1. `1 | x = 1（x为1或者0）`，也就是1可以用于将原值修改为1
2. `0 | x = x（x为1或者0）`，也就是0不会改成原值

我们可以利用这一特性，用于设置二进制中的指定位甚至多位为1
```csharp
Int16 x = 5;
x = (x | 0b0000000000000001); // 将x最后1位设置成1
x = (x | 0b0000000000000011); // 将x最后两位设置成1

// 也可以使用10进制和16进制
x = (x | 1);
x = (x | 0x0001);
```
## 判断两数是否相等
根据异或的规则，“异为1，同为0”，我们可以直接利用这一特性判断两数是否相等
```csharp
bool isSame = (x ^ y) == 0;
```
