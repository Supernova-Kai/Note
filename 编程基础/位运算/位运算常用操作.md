# 左移右移常用操作
### 左移
对于数x，对其左移n位，相当于将其与2的n次方相乘，即：

$$
x << n = x * 2^n
$$

注意，该操作可能会溢出
### 右移
对于数x，对其右移n位，相当于将其与2的n次方相除，即：

$$
x >> n = x / 2^n
$$
# 与运算常用操作
### 取二进制中的部分位
由于与运算必须满足“两个比特位均是1，结果才是1”，我们可以利用这一规则，来提取二进制中的任意一位

假设我们要取x的最后一位，我们只需要将x与1进行与运算即可，这是因为：

1. 对于两数二进制的最后一位，由于1的最后一位是1，如果x的最后一位是1，与运算的结果是1；如果x的最后一位是0，与运算的结果是0，也就是说最后一位的结果取决于x的最后一位是0还是1
2. 对于非最后一位，由于1除了最后一位是1，前面的所有位均是0，由于0与1或者0进行与运算结果都是0，所有不论x前面的值是0还是1，结果均是0，也就是会被丢弃

```csharp
(x & 1) == 1; // 判断最后一位是不是1
(x & 0b0010) == 0b0010; // 判断倒数第二位是不是1
(x && 0b0011) == 0b0011; // 判断最后两位是不是均是1
```

### 判断奇偶性
对于奇数，其二进制最后一位一定是1；对于偶数，其二进制最后一位一定是0

我们可以通过上面的方法，取数的最后一位，来判断其奇偶性
```csharp
// 是否是偶数
public bool IsEven(int n)
{
    return (n & 1) == 0;
}
```
### 清零二进制中的部分位
由于1或者0与0进行与运算结果均是0，我们同样可以利用这一技巧来清零某个数的部分位

不过需要注意的是，需要匹配原数的数据类型（位数）
```csharp
Int16 x = 5;
x = (x & 0b1111111111111110); // 清零x的最后一位

x = (x & 0xFFFE); // 二进制需要写太多0，可以转成用16进制数表示
```
# 或运算常用操作
### 设置二进制中的指定位为1
根据或运算的规则，不论是0还是1，与1进行或运算时，结果均是1；而0和1或者0进行或运算，不会改变原值

我们可以利用这一特性，来设置二进制的指定位为1
```csharp
x = (x | 1); // 1的二进制只有最后一位是1，相当于把x的最后一位置为1
x = (x | 3); // 3的二进制最后两位都是1，相当于把x的最后两位都置为1
```
# 异或运算常用操作
### 判断两数是否相等
根据异或的规则，“异为1，同为0”，我们可以直接利用这一特性判断两数是否相等
```csharp
bool isSame = (x ^ y) == 0;
```
