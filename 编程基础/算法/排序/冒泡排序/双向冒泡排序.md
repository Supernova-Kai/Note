# 双向冒泡排序

其原理和冒泡排序类似，在每一次迭代过程中，从集合的头部和末尾两个方向同时向中间执行冒泡排序，使较大的元素不断向后移动，较小的元素不断向前移动

# 代码
```csharp
public void TwoWayBubbleSort(int[] array)
{
    int start = 0;
    int end = array.Length - 1;

    while (start <= end)
    {
        // 从左到右，把最大的元素推到右边
        for (int i = start; i < end; i++)
        {
            if (array[i] > array[i + 1])
            {
                int temp = array[i + 1];
                array[i + 1] = array[i];
                array[i] = temp;
            }
        }

        end--; // 最右边已经有序，缩小范围


        // 从右到左，把最小的元素推到左边
        for (int i = end - 1; i >= start; i--)
        {
            if (array[i] > array[i + 1])
            {
                int temp = array[i + 1];
                array[i + 1] = array[i];
                array[i] = temp;
            }
        }

        start++; // 最左边已经有序，缩小范围
    };
}
```
